# 给代码添加注释
被注释的代码块在 JavaScript 之中是不会执行的。 在代码中写注释，是一个可以让你自己和以后的其他人理解代码作用的好方法。

JavaScript有两种写注释的方法。

使用`//`注释掉当前行的代码。 这是一个行内注释：
```javascript
// 这是一个行内注释.
```
你也可以使用多行注释来注释你的代码，使用`/*`开始，`*/`结束。 这是一个多行注释：
```javascript
/* 这是一个
多行注释 */
```
**最佳实践**当你写代码的时候，你应该时不时的添加注释来解释你写的代码的作用。 适当的注释能让别人和你未来的自己更容易看懂代码。
